# LocalTwin v0.1 전체 개발 체크리스트

문서 상태: support

이 문서는 LocalTwin v0.1의 완료 품질을 빠뜨리지 않기 위한 확인표다. 작업 순서, 주차, 우선순위와 진행 상태는 [4주 개발 백로그](./tasks.md)에서만 관리한다.

상권 분석 주기능, 데이터 파이프라인, 백엔드/API, 프론트엔드, P1 3D 탐색, 검증과 포트폴리오 기준을 포함한다. 상세 요구사항은 [전체 개발문서](./overview.md)와 기능별 스펙을 기준으로 한다.
